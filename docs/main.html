The Query Object
             		                            
             		           
                               
                            
                 ¶               
               
The query object is used to cache all our filter values. 
Whenever a filter is changed, query is updated.
We can also do the reverse: populate filters based on query's values.
             		                            
  var query = {
    as_of_year: [2012],
    state_abbr: [],
    msamd: [],
    county_name: [],
    census_tract_number: [],
    agency_code: []
  };
